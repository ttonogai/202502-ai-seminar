# AWS AIソリューション ハンズオンセミナー資料

## 概要
Kiro、Quick Suite、GenUの3製品を10分で紹介するプレゼンテーション資料
**注意：公式ドキュメントに基づいて正確な情報に修正済み**

## 構成
- slide01_title.svg - タイトルスライド
- slide02_kiro.svg - Kiro紹介（Agentic AI開発環境）
- slide02b_vibe_coding_handson.svg - バイブコーディング ハンズオン体験紹介
- slide03_quicksuite.svg - Quick Suite紹介（Agentic AIチームメイト）
- slide04_genu.svg - GenU紹介（AWS公式サンプル集）

## 使用方法
各SVGファイルをブラウザで開くか、プレゼンテーションツールで表示してください。
presentation.htmlを開けば、キーボード操作でスライドショーが可能です。
